## Quick Reference

### Generate Components

```bash
# React
codemate g react ComponentName

# Angular
codemate g angular component-name

# Python
codemate g python module_name

# Node.js
codemate g node serviceName

# Java
codemate g java ClassName
```

### AI Agent Commands

```
@component-generator - Generate components interactively
@code-reviewer - Code review assistance
@architect - Architecture guidance
@testing-specialist - Testing strategies
@security-expert - Security recommendations
@devops-engineer - DevOps assistance
```

## File Conventions

| Framework | Naming | Extension |
|-----------|--------|-----------|
| React | PascalCase | .tsx |
| Angular | kebab-case | .component.ts |
| Python | snake_case | .py |
| Node.js | camelCase | .js |
| Java | PascalCase | .java |
